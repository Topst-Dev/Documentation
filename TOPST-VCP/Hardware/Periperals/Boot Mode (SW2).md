<h1 style="color:red">
  Boot Mode (SW2)
</h1>


TOPST VCP has one pin for boot configuration using Boot Mode (BM) and supports 2 modes:
UART boot mode and normal mode.

The slide switch (SW2) is used to select the boot modes of the TOPST VCP.
